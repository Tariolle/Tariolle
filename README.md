AI & Deep Learning engineering student at [INSA Rouen Normandie](https://www.insa-rouen.fr/en). Currently working on an INSA-certified capstone project with Enedis, designing a digital twin of the French electricity distribution network using Big Data pipelines (Spark) and generative Deep Learning models to simulate the load curves of 38 million customers.

## Featured Projects

**[Rose](https://github.com/Alban1911/Rose)** — Open-source League of Legends skin changer seamlessly integrated into the LoL client. Co-led a 6-person team and scaled to **12K+ daily active users**. Architected the full-stack infrastructure, achieving a 10x performance increase by replacing legacy AI components with optimized deterministic systems.

**[SLS-WM](https://github.com/Tariolle/sls-wm)** *(ongoing research)* — Structured Label Smoothing for Discrete World Models: a topology-aware training objective for discrete latent predictors using Gaussian kernels over FSQ codebook structure. Includes DeepDash, a World Model trained entirely in imagination on Geometry Dash and deployed zero-shot on the real game at 30 FPS via screen capture.

**[Opportunistic Target Selection](https://github.com/Tariolle/opportunistic-target-selection)** *(paper submitted)* — A lightweight, architecture-agnostic wrapper that adds early target selection to any score-based black-box adversarial attack, eliminating class drift without requiring gradient access. Includes an interactive demonstrator for exploring adversarial attacks across different networks and images.
